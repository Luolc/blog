CAPU技术组考核项目笔记之一『刹车』

  • CAPU
  • 修车
  • 时间线：第三教学周，春训第一周
  • 概述：原理、理论考核问题整理
  • TO-DO：来令片更换详解(2015技术组实践考核要求)、V刹闸皮更换详解(2016技术组实践考核要求)

原理

碟刹

碟刹部分主要来源于果粒橙的2015年技术组考核碟刹部分资料。传送门
调碟是日常拉练、出摊中经常要做的，无论从考核还是实践上都应当熟练掌握。
2015考核的25min时限非常宽裕，日常练习时可以自己计时感受一下。尽量做到10min以内完成。

零、夹器 & 来令片

0.0 常见分类
按作用介质分为油压式，机械式。

  • 油压式：俗称油碟，靠油的液压来刹车。
  • 机械式：俗称线碟，靠线拉。

按作动方式分为双边作动和单边作动。

碟刹靠固定在车架上的来令片夹固定在车轮上的碟片制动。来令片有左右两片。双边作动碟刹刹车时两边来令片同时往中间移动夹住碟片；单边作动碟刹刹车时只有一片来令片动(通常是外侧)，挤压碟片使碟片形变，与另一侧来令片一起夹住碟片制动。
油碟都是双边作动；线碟多是单边作动。双边作动优点：碟不容易瓢。本文中如无特殊说明，默认油碟=双边作动，线碟=单边作动

0.1 夹器的固定
夹器的固定位置可以调整。大多靠两个夹器固定螺丝确定。夹器上的螺丝孔比螺丝大，可以在螺丝松一点后左右移动。夹器位置需要调整至来令片与碟片平行，并且碟片两侧都留有合适大小的空隙。


上下两端的孔可以插入固定螺丝，注意到孔不是圆形的，可以在长轴方向移动，来调整来令片和碟片的相对位置。

0.2 来令片
来令片相当于V刹的闸皮，当刹车介质磨损到0.5mm后需要更换。
左右两块来令片通常被放置在一个金属框架的弹簧上。

0.3 碟刹不回弹
首先检查来令片之间的金属弹簧是否损坏。

线碟不回弹可能因为线管涩，清线管解决；若夹器故障（少见），修/换夹器。

油碟不回弹大多因为油碟进气。解决方法：1.去车店换油 2.直接换刹车
油碟在没有进气时可以把车倒置；进气后倒置会使得空气上浮，使得回弹困难。如果进气量不大，气体可以上升至闸把处。若气泡进入夹器，会导致回弹困难，猛捏几次刹车有可能解决。也可以尝试用组合或改锥敲开合上后无法回弹的来令片。尽快去车店换油/夹器。

一、碟片

油碟线碟的碟片通用。

1.0 常见规格

  • 按直径分：

140mm, 160mm(常见), 180mm(AM(全山)车), 203mm(DH(速降)车)。

直径越大，刹车时力臂越大，刹车效果越好。

  • 按固定方式分:

1.六钉碟片

固定多用T25梅花扳手，安装时要小心，不要把螺丝上花了。安装时注意交替上紧，最终上紧的顺序参考《M416用户手册》(下图)。

2.中锁碟片 如下图


优点：容易安装，不容易瓢，刹车效果好(类似卡飞相比旋飞的优势)。缺点：有点点贵

1.1 安装
安装：注意方向，大多数有箭头指示，大多数为印字一面向外。

六颗螺丝钉先较松的安装上，再交替上紧，具体顺序见图，扭力必须相同(用手掌握即可。。。扭力扳手比较贵)。必须上紧，孔比螺钉直径大，不上紧会有框量，会瓢。

1.2 磨合

1.3 保养
托运时，可以把夹器、碟片拆下随身带。

保养：一般不用保养。把尘土擦干净就好，或者可用酒精；碟片不能沾油，沾油后需和来令片一起更换；也可用汽车刹车清洗剂清洗碟片。清洗后需要重新磨合。

1.4 常见问题

1.4.0 碟瓢
大多因为意外撞击或六钉碟片上紧时每个螺丝扭力不同。
解决办法：1.更换。2.掰，重新安装。掰碟片可以用手(大多数情况可以掰好)，或用掰碟片工具。大多数情况可以直接掰好。

1.4.1 刹车时异响
常见原因：
1.碟片上沾有不明液体，刹几次就好了。
2.碟片没有磨合好
碟片磨合原理见1.2。如果磨合失败，印层没有均匀敷在碟片表面，而是从侧面看类似波的形状敷在碟片表面，刹车时就会发出异响。对使用影响不大。解决方法：用汽车刹车清洗剂清洗碟片和来令片之后重新磨合。

1.4.2 碟片t25螺丝钉被上花
在将没有上花的部分砸出两道凹槽，用一字改锥取出(操作难度大，建议去车店)。

P.S. 押后考时拿龙部分，由于正好有一辆车断了一根辐条，Bee司令就安排我跟师姐安装一根新的上去。拆到碟片上剩下最后一个螺丝时才发现已花，找来很多老会员帮忙拆都无果。即使按照砸出凹槽用一字改锥的方式强行取下，安装时也没有替代品，所以基本上还是能去车店解决最好。另有司令语『先把每个螺丝都拧松确定没有已经花的，省的拆到最后一个花了蛋疼』

二、调碟

常见的蹭碟有如下几种情况。

2.0 轮歪/龙
当发现轮蹭时，一定先看轮。重装或拿龙。

P.S. 八达岭拉练行进至清华北门左右时一车蹭，后旗小分队到达时队中押后(我们萌萌的傻枫)告知碟已调好，扎紧码表磁头即可。后旗小分队愉快的准备上路时，果粒微笑着把我拉到车前道：“仔细找找有没有问题。”待我定睛一看猛然发现……轮歪，碟蹭前叉(手动再见)。遂被师祖教育『先看轮子』×3.

2.1 碟瓢
见1.4.0

2.2 夹器位置不对
油碟调法：松开固定螺丝 → 捏紧刹车 → 上紧螺丝
线碟调法：松开固定螺丝 → 目视调整

2.3 闸程不合适
油碟闸程太短可以放油(不推荐自己尝试，去车店)。
单边作动线碟闸程可以靠微调旋钮和闸线长度来调整。


调节微调旋钮可以改变来令片之间初始距离，顺时针旋距离变短。


夹线器。可以通过闸线粗调和微调来调整闸线长度。

2.4 不回弹
见0.3

2.5 调整时不蹭，骑行时后碟蹭
可能快拆未上紧。
大部分情况是由于调整时车重心靠脚支一侧，车架变形，使得碟片更加靠外，正常骑行时碟片蹭内侧来令片。可以在调整时在内侧都留出一些空隙。

V刹

V刹大部分是我根据技术文档和自行拆装的学习感受，因为V刹是自学的，并不太系统。
少许参考车友版老贴和2012年修车讲座视频。

零、结构


参考：Shimano B-RM770 技术文档

0.0 概述
V刹依靠闸皮与车圈的摩擦来制动。闸皮有左右对称的两块，固定在两侧闸瓦中部，闸瓦通过下方的连杆固定螺栓(上图中①)固定在车架上。
收紧闸线时，线拉闸瓦上端，以两个固定端为轴向内侧转动，闸皮因此贴近车圈制动。
松开闸线时，平行闸瓦的两根铁丝提供使闸瓦向外侧转动的张力。铁丝的初始位置可以通过张力螺丝(上图中②)调整。

0.1 V刹的固定
车架的弹簧窝上有三个小孔可以插入，选择不同的孔弹力会相差很多。没有特殊偏好的话选择中间的即可，弹力比较适中。
插入后旋紧固定螺栓即可。越紧越好，性命攸关的部位不可疏忽。

0.2 闸皮
闸皮经过一组共5个垫圈，固定在闸瓦上。垫片用于调节闸瓦的凸起高度，需要保证闸瓦上端相距不要过短。
闸皮纹路基本磨皮后，需要进行更换。若闸皮内的白铁皮露出，继续使用会损害车圈。

一、调闸

1.0 轮歪/龙
当发现轮蹭时，一定先看轮。重装或拿龙。

1.1 张力铁丝形变程度不同
刹车使用一段时间后，两侧的铁丝变形程度可能不同，导致刹车不对称，严重时其中一边闸皮会蹭车圈。
不妨设左侧闸皮蹭，此时旋紧左侧/旋松右侧张力螺丝都可以使得刹车向正确位置移动。注意两个螺丝协调，不要光旋一侧。当调整至两个闸皮对称，捏闸时同时作动并同时接触车圈为好。如果只调节张力螺丝已经无法完全调节对称，有可能是铁丝形变过大，可以将铁丝卸下掰一掰，但是较难把握，一般不需要。

1.2 闸程不合适
通过闸线的粗调螺母和闸把上的微调螺母可以调节闸程。
闸皮使用一段时间过后，会磨损一部分，使得闸程变长。同时，长时间使用刹车后闸线会变长，同样导致闸程变长(与碟刹同理)。这时一般旋松微调螺母即可。

1.3 闸皮位置不对/磨损
在拉紧刹车时，闸皮应当尽量与车圈贴合。
闸皮不应过高或过低。过高会蹭外胎，长期磨损外胎会导致外胎爆裂。过低刹车效果差，废闸皮，同时有可能导致闸皮磨损不均匀。
当闸皮没有与车圈平行时，也会导致磨损不均匀。这种闸皮磨损面较光滑，贴合车圈时排出内部空气，由于大气压的缘故，会对车圈产生一种吸力，使得闸不回弹。

通过旋转闸皮与闸瓦间的垫片可以调整闸瓦上端距离。

闸皮如果磨损到露出白铁，刹车效果变差，同时磨损车圈。长期以往车圈会断。此时应尽快更换闸皮。

1.4 闸线管弯折/堵塞
拆闸线，清理。将线管整理到合适的位置。
弯折的太厉害就换新的。

1.5 三小孔的选择
两边实在调不对称，可能是小孔装的不一样(几乎不会遇到，除非技术组考核特地挖这个坑)。
如果感觉弹力特别不合适，可以尝试换小孔，一般也不会用到。

理论考核提纲

碟刹

  • 分析蹭闸原因
    轮歪/龙；夹器位置不对；闸程太短
  • 线碟结构
    见前文和技术文档。
  • 油碟不回弹原因；油碟车可否倒置？
    油碟进气；进气后倒置会使得回弹困难。
  • 碟片安装方法(六钉碟片、中锁碟片)、注意事项
    见技术文档。
  • 碟刹车轮左右辐条编织方式相同吗？
    后轮相同，前轮左右不同。
  • 碟片瓢了怎么办？
    换/掰。
  • 空车调碟时不蹭，骑上去时蹭是为什么？怎么办？
    (即碟刹2.5部分)
    可能快拆未上紧。
    大部分情况是由于调整时车重心靠脚支一侧，车架变形，使得碟片更加靠外，正常骑行时碟片蹭内侧来令片。可以在调整时在内侧都留出一些空隙。
  • 闸程调整至怎样比较合适？
    过长或过短均不合适。个人习惯闸把在与车把基本平行时抱死为好。前闸闸程应略大于后闸。

V刹

  • 如果V刹偏向左边，导致右边的闸皮出现蹭闸，应如何调节弹力螺丝？
    松左侧螺丝，紧右侧螺丝。
  • 如果调节弹力螺丝对调节弹力已经无能为力，还可以用什么办法调节弹力？
    车架上装闸的小孔有三个，改变装闸的小孔可以调节弹力。另外，掰那根作为弹簧的铁丝应该也可。
  • 装闸皮时如何保证闸皮在刹车时能紧贴车圈？
    握住闸瓦使其贴近车圈，旋紧闸瓦固定螺帽。
    闸皮距离外胎约1mm.
  • 闸皮位置太高、太低分别会有什么坏处？有时捏闸放开后闸弹不回来，为什么？
    太高会磨到车胎，严重的会导致爆胎。即使不严重的，久而久之应该也会爆胎；太低让闸皮磨损不均匀，影响刹车效果还废闸皮。
    有时捏闸放开后闸弹不回来，就是因为闸皮磨损不均匀产生闸皮对车圈的吸力。也有可能是线管涩/弯折、闸线绣。
  • 闸皮磨损过多有何坏处？一般何时需要更换闸皮？
    闸皮磨损过多导致刹车效果降低，而且闸皮里的钢铁部分与车圈磨擦会损毁车圈。有的车有装挡泥板，如果V闸形状是直的捏闸时可能会蹭到挡泥板，会同时损毁挡泥板跟V闸。一般到闸皮上的纹路基本磨平，可以看到里边的白铁片时应该换闸皮。
  • 与闸皮相连的柄上那些垫片有什么用处？
    调节闸瓦凸起的高度，改变闸瓦上端的距离。(具体作用待查)
  • 如何把闸线从闸把上拆下来？
    调节微调螺母、微调锁紧螺帽与闸把三管一线。沿着侧向的槽即可取出闸线，再捏闸卸下卡头即可。
  • 闸把角度应该是怎样的？
    向下倾斜40°左右。
  • 如何调节闸程（闸的松紧）？微调螺丝扭出来一厘米可以吗？
    调节闸线的粗调螺母或闸把上的微调螺母。
    调出太多感觉不好，至少容易锈。(待查)
  • V刹有哪些优点？
    质量轻、便宜、结构简单易于安装和维护。
  • 好的手感应该是怎样的？如何让一辆车的闸有更好的手感？
    有合适的闸程；有合适的弹力；阻力小，松开闸后可以迅速回弹。
    定时清线管；按需调整闸程；需要时及时更换闸皮。
  • 如果发现闸的刹车效果不明显了，可能是什么问题，怎么办？
    轮歪；车圈龙；车圈上有污物。
    线管涩/弯折；闸线锈。
    闸两侧不对称；闸程过长；闸皮位置不对/闸皮需要更换。
